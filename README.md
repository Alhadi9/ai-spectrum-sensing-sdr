# AI-Powered RF Spectrum Sensing

## Overview

This project presents an Artificial Neural Network (ANN)-based spectrum sensing system for cognitive radio networks using Software Defined Radio (SDR) and real USRP hardware.

## Objectives

* Detect wireless signals under low-SNR conditions.
* Improve spectrum utilization efficiency.
* Evaluate AI-based signal detection on real hardware platforms.

## Technologies

* MATLAB
* Artificial Neural Networks (ANN)
* Software Defined Radio (SDR)
* USRP
* Signal Processing
* MIMO Systems

## Key Results

* Detection probability of 88.6% at -11 dB SNR.
* Detection probability of 80.9% at -13 dB SNR.
* Detection probability of 63.3% at -15 dB SNR.
* Improved detection performance to 84% using 8 synchronized antennas.

## Hardware Platform

* USRP SDR devices
* Multi-antenna MIMO configuration
